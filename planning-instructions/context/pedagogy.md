# Pedagogy — read before planning

## The framework (full version: Edtech vault → Frameworks/Vision/Framework Core)

1. Thinking first, tool second: learning purpose → thinking strategy → student role → (optional) technology → artefact → evidence.
2. The removal test: if the technology's output were removed, would visible student thinking remain? iPad form: "Why not paper?"
3. Every task leaves an assessable trace of thinking the tool did not produce.
4. Applies to the application layer only — explicit teaching of foundational skills (phonics, number facts, technique) is direct instruction territory.

- Success criteria co-constructed where practical; returned to mid-lesson and at reflection.
