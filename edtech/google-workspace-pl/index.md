## Summary

This folder holds a five-session Google Workspace for Education PL pack for Years 3-6 iPad implementation. It includes teacher session plans and paired cheat sheets for Google Classroom, assignments, feedback, Forms quizzes, and Forms data analysis.

## Session Sequence

| Session | Focus | Useful for |
|---|---|---|
| 1 | Create a roster in Google Classroom | Helping teachers set up classes and get students joined. |
| 2 | Assign work and collect assignments | Building basic Classroom workflow and submission routines. |
| 3 | Give and receive digital feedback | Supporting timely feedback, private comments and rubrics. |
| 4 | Create surveys and quizzes in Google Forms | Exit tickets, reflection forms, self-marking quizzes and unit checks. |
| 5 | Use Google Forms to track student progress | Turning responses into planning evidence through summaries and Sheets. |

## Coaching Use

- Use the sessions as a scaffold for PLC-based micro-PL rather than a one-off workshop.
- Pair each staff PL session with an in-class implementation visit so teachers try the workflow with students quickly.
- Connect Forms data back to PLC evidence cycles and reporting conversations.
