# Tinkercad — Classroom Guide

**Tinkercad** is a free, browser-based app by Autodesk for 3D design, electronics, and coding. It's suitable for primary students and has strong classroom management features via Tinkercad Classrooms.

---

## Summary

Tinkercad Classrooms lets teachers set up a managed class environment where students sign in via a class code (no personal email required for under-13s). Teachers get a dashboard to view, moderate, and edit all student designs. Safe Mode is on by default, restricting public sharing and distracting content. Free to use; integrates with Google Classroom.

---

## Getting Started

### Set your account role to Teacher
- Log in → click your avatar (top right) → **Settings** → **Edit profile** → change **Role** to **Teacher** → Save
- Once you create your first class, your role is locked as Teacher (you become a moderator for student accounts)

### Create a class
- From the Classes dashboard, click **Create new class**
- Enter: class name, age/grade level, subject
- Choose whether Safe Mode is on by default (recommended for primary)
- Max 250 students per class

## Student Onboarding Options

| Method | Best for | Notes |
|---|---|---|
| **Google Sign-On + Class Link** | Students with Google accounts | Share link; students log in with Google — no Tinkercad account needed |
| **Login Code + Class Code** | Students under 13, no email | Teacher creates roster; students get a nickname + code slip. No email required. |
| **Email + Class Code** | Students 13+ with existing accounts | Standard login |

> **For primary school (under 13):** use Option 2 (Login Code + Class Code). Students don't need an email address — just a slip of paper with their nickname and the class code.

## Class Dashboard Tabs

| Tab | What it does |
|---|---|
| **Students** | Add/remove students, manage Safe Mode per student |
| **Activities** | Create and assign design tasks with starting templates |
| **Designs** | Bird's-eye gallery view of all student work (3D, Circuits, Codeblocks) |
| **Moderation** | Detailed activity log — flag or ban inappropriate content |
| **Co-teachers** | Invite up to 5 co-teachers to view/moderate the class |

## Safe Mode

Safe Mode is **on by default** for all students under 13 who join a class. It:
- Limits gallery/search to curated Staff Favourites only
- Prevents students from making designs public
- Disables social sharing, profile images, and social links
- Hides non-featured community shapes in the shape library

**Teachers can:**
- Toggle Safe Mode for the whole class or individual students
- Make a specific student's design public while keeping them in Safe Mode (useful for showcases)

> Safe Mode can only be disabled by the teacher (or parent/guardian for personal accounts).

## Google Classroom Integration

- Install the [Tinkercad Add-on for Google Classroom](https://workspace.google.com/marketplace/app/autodesk_tinkercad/614288788007)
- Supports due dates, review, and grading within Google Classroom
- Students under 13 still need to be onboarded via Tinkercad class code first
- Teachers can send Tinkercad designs as Google Classroom assignments and receive student submissions

## Student Privacy (COPPA / Australian context)

Tinkercad is a US-based Autodesk product. Under COPPA, schools can act as the parent's agent to consent to data collection for children under 13, **for educational purposes only**.

**What Tinkercad collects from students:**
- Username and password, birthdate, country
- Designs and content created in the app
- Browser/device info, IP address

**What Tinkercad does NOT do:**
- Sell student data to third parties
- Use student data for advertising
- Share beyond the teacher and trusted service providers

> The Education Provider (teacher/school) accepts the service agreement on behalf of parents. It's good practice to inform parents — this notice can be shared with them.

**Contact for privacy:** tinkercad.support@autodesk.com

## Common Issues

| Problem | Fix |
|---|---|
| Students get error opening shared design | Design isn't public — go to design settings and make it public |
| Can't make your own design public | You've accidentally joined your own class and are in Safe Mode — remove yourself from the class |
| Can't open student's submitted design | Student used a different account — browse via Class Gallery instead |
| Class finished — want to preserve student work | Archive the class (don't delete) or help students graduate their seat to a personal account |

---

## Archiving vs Deleting

- **Archive:** moves class out of main list; student work preserved; students can still access work; teacher must still moderate
- **Delete:** permanently deletes all student seats and associated designs — use with caution

---

## Using Activities (Assigning Work to Students)

Activities let you give students a starting design template to copy and build from, keeping their work organised by task.

**How to create and assign an Activity:**
1. Open the class → go to the **Activity tab**
2. Click **"New Activity"** → add a title and description
3. Create a new design from within the Activity, OR go to any existing design on your dashboard → gear icon → **"Add to Class Activity"**

**Student experience:**
- Students see the Activity and click **"Copy and Tinker"** to start their own version
- Students can also add their own existing designs to an Activity via the gear icon

**Teacher benefits:**
- Only see designs students created for that specific Activity — no sifting through everything
- Deleting an Activity doesn't delete student designs (they stay in each student's profile)
- Works across all 3 Tinkercad spaces: **3D Designs, Circuits, and Codeblocks**
- Activities can be **copied between classes** — reuse the same Activity setup next year

---

## Tips for Primary School Use

- Use **Login Code + Class Code** (Option 2) for all students under 13 — no email headaches
- Keep **Safe Mode on** for all primary students
- Use the **Designs tab** for a quick gallery walk during or after lessons
- Use **Activities** to give students a starting template (avoids blank-page paralysis)
- You can **Tinker a student's design** directly from the Students tab — great for live help
- Assign **Design & Make skill badges** as recognition for completed work
