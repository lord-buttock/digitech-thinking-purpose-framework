# Vision Framework

> A unified technology integration framework built on the principle that technology should earn its place by making student thinking more visible, not just more engaged.

## Core Thesis

**Meaningful iPad Use** and **PTCCD** are the same framework built twice — one for iPads, one for AI. The Vision framework unifies them under a single evaluative lens: the **Removal Test**.

> *"If you removed the technology from this task, would the thinking be diminished or just the output?"*

## The Five Principles

1. **Thinking first, tool second** — technology supports cognition, not replaces it
2. **The Removal Test** — if the tech disappeared, would the quality of thinking still be evident?
3. **Assessable trace** — the technology must leave an artefact that shows student reasoning
4. **Named failure mode** — each framework instance has a specific wrong use to avoid
5. **Application layer only** — technology operates at the enhancement layer, not the core learning layer

## Two Framework Instances

### Meaningful iPad Use (mature)

The older of the two instances. Applied to iPad use in primary classrooms. Well-developed, tested in classroom context.

### PTCCD (draft)

Applied to AI use in learning tasks. Pattern: **Position → Tension → Consequence → Challenge → Defence**. Students use AI as an intellectual sparring partner, not an answer machine.
