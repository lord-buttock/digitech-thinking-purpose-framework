# Meaningful iPad Use

> Not more iPad. Better iPad.

This folder contains the teacher-facing framework notes for meaningful, collaborative and creative iPad use in primary classrooms.

The purpose of this framework is to help teachers move students from passive screen use toward active creation, shared thinking, evidence capture, explanation, collaboration and curriculum-aligned digital artefacts.

## One-Screen Quality Test

Use iPads when they help students capture evidence, replay, zoom, record voice, annotate, revise, compare, model, simulate, publish, collaborate or explain to an audience.

Use paper, whiteboards, manipulatives, talk or books when those methods serve the learning just as well or better.

The planning logic is:

```text
Unit phase -> learning intention -> thinking strategy -> digital action -> student artefact -> evidence of learning
```
