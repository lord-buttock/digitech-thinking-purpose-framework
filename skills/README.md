# Education Agent Skills

164 skills for Claude's plugin system, created by Gareth Manning (open source).
Source repository: https://github.com/GarethManning/education-agent-skills

## How to use

**Find a skill by planning task:** [quick-finder.md](quick-finder.md)
**See all skills by domain:** [index.md](index.md)
**Browse by domain:** [domains/](domains/)

## How to invoke

Ask Claude: *"Use the `skill-name` skill to…"*

Example: *"Use the `backwards-design-unit-planner` skill to plan a Year 3 persuasive writing unit."*
