# Curriculum Alignment

Audits, maps, and crosswalks learning objectives and content descriptions against curriculum frameworks.

## Skills in this domain (4)

| Skill | What it does |
|---|---|
| [[Coverage Audit]] | Take a curriculum framework and a statutory or accreditation requirement list and produce a coverage table, gap summary, and CSV showing which framework content covers each requirement and where gaps exist. Use when checking whether a programme covers everything a curriculum authority or accreditor requires. |
| [[Curriculum Crosswalk]] | Compare two or more curriculum frameworks and produce a topic matrix showing coverage and gaps across all of them, plus an optional reference-centric crosswalk document for PLC use. Use when comparing a school framework against state standards or mapping overlap between curriculum documents. |
| [[Developmental Band Translator]] | Tag decomposed curriculum items (KUDs, learning targets, criteria) with a school's developmental band metadata while preserving the source's voice and labels. Use when re-labelling curriculum content with your school's band system; supply the band schema and the skill derives mapping rules from it. |
| [[KUD Chart Author]] | Author or review Know/Understand/Do charts for competency-based learning targets across developmental bands, working from raw curriculum documents through to existing learning target sets. Use when writing or quality-checking KUD charts for a unit, competency, or programme. |
