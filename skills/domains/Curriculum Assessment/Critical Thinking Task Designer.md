# Critical Thinking Task Designer

Design discipline-specific critical thinking tasks grounded in knowledge-contingent reasoning rather than generic skills. Use when embedding higher-order thinking into subject content. For literacy- and media-focused analysis tasks (evidence, bias, arguments), use critical-analysis-task-designer.
