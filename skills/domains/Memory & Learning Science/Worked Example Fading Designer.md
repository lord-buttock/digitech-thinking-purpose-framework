# Worked Example Fading Designer

Design a worked example fading sequence from fully worked examples through to independent practice. Use when teaching procedures, algorithms, or multi-step processes to novice learners. If students already follow examples but cannot solve problems alone, use worked-example-to-problem-solving-transition-designer.
