# Interleaving Unit Planner

Redesign a blocked topic sequence into an interleaved plan with mixed practice across related topics. Use when mixing practice across topics within units, homework schedules, or revision programmes. For scheduling when topics are revisited over time, use spaced-practice-scheduler.
