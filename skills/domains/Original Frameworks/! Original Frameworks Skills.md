# Original Frameworks

Orchestration-level skills that synthesise multiple frameworks: rubric design, compassionate systems, regenerative inquiry, transformative learning.

## Skills in this domain (17)

| Skill | What it does |
|---|---|
| [[Assessment Design Orchestrator]] | Routes between five assessment pathways — formative, rubric/criteria, authentic/performance, peer/self, and diagnostic — with validity and equity checks. Use when a teacher needs help choosing how to assess. |
| [[Coherent Rubric Logic Builder]] | Build a five-level rubric with coherent logic for a learning target within a developmental band. Use for Manning methodology programmes where Competent = success. For general curriculum rubrics, use criterion-referenced-rubric-generator instead. |
| [[Compassionate Systems Awareness Orchestrator]] | Orchestrate compassionate systems tools from issue or aspiration through to wise action. Use when a class is investigating a real-world issue or imagining a better future and needs a complete inquiry workflow from understanding the system to taking action. |
| [[Developmental Band System Designer]] | Design a developmental band system mapping student growth from early childhood through upper secondary. Use when building competency-based curriculum architecture for a school or programme. |
| [[Developmental Progression Synthesis]] | Synthesise completed KUD charts into a developmental progression matrix and per-competency narrative sections. Use when you need a programme-level view of how knowledge, understanding, and performance develop across bands. |
| [[Dilemma Navigation For Education Design]] | Identify and navigate genuine dilemmas in curriculum, school, or community contexts — tensions between competing goods that cannot be solved, only navigated — producing a structured dilemma map with both poles named and both/and possibilities. Use when a planning conversation keeps circling a tension with no clean answer, like depth vs. coverage or autonomy vs. support. |
| [[Dispositional Knowledge Assessment Designer]] | Design multi-informant assessment approaches for dispositional competencies like curiosity or resilience. Use when assessing character strengths or competencies that written tests cannot capture. |
| [[Inclusive Design Orchestrator]] | Coordinates UDL and differentiation tools through a universal-first hierarchy: barrier removal before targeted differentiation before individualised accommodation. Use when planning accessible learning. |
| [[Learning Target Authoring Guide]] | Author learning targets for a competency across developmental bands with precise, observable progression language. Use when writing 'I can' statements for competency-based programmes. |
| [[Multi Perspective Decision Wheel]] | Structure a decision or design challenge through multiple perspectives before committing to action. Use when a group faces a decision and wants a wiser next step — works best as a synthesis step after an inquiry has been scoped and mapped. |
| [[Place Based Curriculum Orchestrator]] | Present pathway options and orchestrate place-based curriculum design from a local place, curriculum requirement, or community issue — routing between place-based-inquiry-anchor, ecological-inquiry-anchor-designer, and outdoor-learning-sequence-designer. Use when place should become a primary text for learning. |
| [[Regenerative Project Design Orchestrator]] | Present project-design pathway options and orchestrate regenerative projects using backwards design, PBL, SEEDS, compassionate systems action, or civic/service pathways. Use when planning a sustainability, community, or regeneration project and choosing the right design approach. |
| [[Scoping For Transformative Learning Inquiry]] | Define the scope and purpose of a complex inquiry before mapping or futures work begins. Use when a class or group needs to agree what they are investigating, who it matters to, and at what scale. |
| [[Seeds Regenerative Inquiry Cycle]] | Design a SEEDS regenerative inquiry cycle connecting place-based learning to ecological awareness for young learners. Use when building early childhood or primary inquiry around ecosystems and community. |
| [[Self Determined Project Design Protocol]] | Structure a student's self-determined project with mentoring checkpoints, competency alignment, and reflection. Use when guiding student-led projects within a competency-based framework. |
| [[Single Point Rubric Designer]] | Design a single-point rubric with one criterion and open columns for evidence. Use for student self-assessment, peer feedback, teacher formative feedback, or pre-task planning. Works with any learning target, with or without a band system. |
| [[Three Horizons Learning Transition Mapper]] | Map a scoped topic across three coexisting horizons — the current system under strain, the preferred future, and the transition innovations between them — to understand change dynamics and identify responsible next actions. Use when a class is exploring how a system could change over time, typically after scoping an inquiry. |
