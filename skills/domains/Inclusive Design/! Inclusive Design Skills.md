# Inclusive Design

Applies Universal Design for Learning (UDL) principles to audit, design, and improve lessons for all learners.

## Skills in this domain (3)

| Skill | What it does |
|---|---|
| [[UDL Barrier Anticipator]] | Predict access barriers in a learning task before delivery, given a learner variability profile — distinguishing barriers addressable through design from those requiring specialist support. Use when checking an upcoming task for accessibility or planning for a class with diverse learning needs. |
| [[UDL Lesson Auditor]] | Audit an existing lesson against UDL's three principles — engagement, representation, and action/expression — identifying specific access barriers and concrete modifications ranked by impact. Use when reviewing a lesson for accessibility or making an existing lesson more inclusive. |
| [[UDL Options Designer]] | Generate multiple means of engagement, representation, and action/expression for a learning goal — specific, practical alternatives, with the highest-impact single change recommended. Use when building flexible options into a lesson so all students can access the learning. |
