# Critical Analysis Task Designer

Design a critical analysis task targeting transferable skills like evaluating evidence, identifying bias, or analysing arguments in texts and media. Use when embedding critical analysis into literacy, media, or source-evaluation lessons. For discipline-specific critical thinking grounded in subject knowledge, use critical-thinking-task-designer.
