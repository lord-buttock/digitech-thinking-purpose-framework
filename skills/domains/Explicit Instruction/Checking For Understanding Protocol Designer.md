# Checking For Understanding Protocol Designer

Design a checking-for-understanding protocol with specific techniques for each lesson stage. Use when planning systematic comprehension checks during explicit or direct instruction. For a single high-stakes diagnostic question at a key decision point, use hinge-question-designer.
