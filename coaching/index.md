# Coaching

Coaching is a core part of the [[Learning Specialist/Digital Technology Role|Digital Learning Specialist]] role — working alongside classroom teachers across all year levels to embed high-impact, digitally-integrated teaching practice.

**Related:** [[Learning Specialist/Digital Technology Role]] · [[wiki/topic-high-impact-teaching-strategies]] · [[wiki/topic-digital-learning-specialist-role]] · [[wiki/topic-inquiry-based-learning]]

Phill coaches: Foundation, Year 1, Year 2, Year 3, Year 4, Year 5, Year 6.

## Structure

Notes are organised by year level. Each session note is named `YYYY-T[term]-[brief-topic].md` within the relevant year level folder. This way the same folder accumulates across years, making it easy to see patterns within a cohort level over time.

## Session Note Template

Each coaching note should include:

```
---
title: Coaching — [Year Level] — [Topic]
type: topic
tags: [coaching, year-N, term-T]
date: YYYY-MM-DD
term: T[1-4] YYYY
---

## Context
What prompted this session / what were we working on?

## Observations
What did I notice about teaching practice?

## Strategies discussed
What did we talk about trying?

## Follow-up
What to check on next time / action items?
```
