# English Writing Throughlines F–6

**Source:** [[Education/Writing/English Writing Throughlines F-6.pdf]]  
**Origin:** School-created curriculum planning document  
**Scope:** Foundation to Year 6, across 7 writing trait domains

---

## Summary

A comprehensive F–6 writing progression document covering 7 writing trait domains. For each domain and each year level, it specifies Key Understandings, Key Skills, and Essential Vocabulary. This is a core curriculum planning tool for English writing — used to sequence teaching, plan writing units, and benchmark student progress against school-agreed expectations.

---

## The 7 Writing Trait Domains

| Domain | What it covers |
|---|---|
| **Sentence Fluency** | Sentence variety, rhythm, flow, transitions |
| **Word Choice** | Precise, vivid, appropriate vocabulary selection |
| **Presentation** | Layout, handwriting, formatting, visual organisation |
| **Conventions** | Spelling, punctuation, grammar |
| **Ideas** | Content, detail, focus, originality of ideas |
| **Audience** | Awareness of reader, purpose, register, tone |
| **Organisation** | Structure, sequencing, openings, conclusions |

---

## How to Use

- **For planning:** Identify the year level and domain focus for a writing unit → use Key Understandings to set learning goals, Key Skills for success criteria
- **For moderation:** Compare student work against the Key Skills descriptors for the relevant year level
- **For reporting:** Use Essential Vocabulary and Key Understandings to write accurate report comments
- **For coaching:** Reference when supporting teachers to plan writing instruction or sequence writing traits across a term
